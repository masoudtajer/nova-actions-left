# Nova Actions Left

Moves the resource actions column to the left on the ResourceTableRow.

![Example](screenshot.png)

## Installation

You can install the package via composer:

For Laravel Nova 5+:

`composer require saintsystems/nova-actions-left`

For Laravel Nova 4.x and below, see [saintsystems/nova-actions-left v4.0.2](https://github.com/saintsystems/nova-actions-left/tree/v4.0.2).

The package will automatically register itself.
